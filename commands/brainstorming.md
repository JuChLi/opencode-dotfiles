Turn ideas into fully formed designs through collaborative dialogue.

## Instructions

Load the `brainstorming` skill and follow its design exploration process.

$ARGUMENTS

## Steps

1. Use the Skill tool to load the `brainstorming` skill
2. Follow the checklist:
   - Explore project context (files, docs, recent commits)
   - Ask clarifying questions one at a time
   - Propose 2-3 approaches with trade-offs
   - Present design in sections, get approval
   - Write design doc to `docs/plans/`
   - Transition to `writing-plans` for implementation
3. Do NOT write any code until design is approved
