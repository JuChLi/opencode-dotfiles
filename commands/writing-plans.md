## Instructions

Load the `writing-plans` skill and create a comprehensive plan.

$ARGUMENTS

## Steps

1. Use the Skill tool to load the `writing-plans` skill
2. Create plan with:
   - Header: Goal, Architecture, Tech Stack
   - Tasks broken into 2-5 minute steps
   - Exact file paths (create/modify/test)
   - Complete code snippets
   - Test commands with expected output
   - Commit messages
3. Save to `docs/plans/YYYY-MM-DD-<feature-name>.md`
4. Offer execution choice: Subagent-Driven or Parallel Session
