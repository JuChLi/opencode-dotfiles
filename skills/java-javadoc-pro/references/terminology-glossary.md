# Java Javadoc 術語表（zh-TW）

本文件用於統一 `java-javadoc-pro` 相關文件與輸出文字的術語。

## 常用術語

- 風格設定檔（style profile）
- 內建風格（built-in style）
- 自訂風格（custom style）
- 初學者可讀模式（beginner readability mode）
- 風格來源（style source）
- 掃描（scan）
- 產生（generate）
- 精修（refine）
- 品質檢查（lint）
- 品質閘門（quality gate）
- 覆蓋率（coverage）

## Javadoc 欄位術語

- 摘要首句（summary sentence）
- 參數說明（`@param` description）
- 回傳說明（`@return` description）
- 例外拋出說明（`@throws` description）

## 非同步語意術語

- 非同步完成語意（async completion semantics）
- `Future` 成功完成（successful completion）
- `Future` 失敗完成（failed future）

## 建議語氣

- 使用台灣繁體中文。
- 優先使用「回傳」「拋出」「逾時」「流程」「語意」等用詞。
- 避免低資訊句型（例如「操作結果」「方法輸入參數」）。
