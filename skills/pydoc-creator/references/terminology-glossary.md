# Python Docstring 術語表（zh-TW）

本文件用於統一 `pydoc-creator` 相關文件與輸出文字的術語。

## 常用術語

- 文件字串（docstring）
- 模組文件字串（module docstring）
- 類別文件字串（class docstring）
- 函式文件字串（function docstring）
- 方法文件字串（method docstring）
- 風格設定檔（style profile）
- 內建風格（built-in style）
- 自訂風格（custom style）
- 掃描（scan）
- 產生（generate）
- 精修（refine）
- 品質檢查（lint）
- 品質閘門（quality gate）

## Python 文件欄位術語

- 摘要首句（summary sentence）
- 參數說明（parameter description）
- 回傳說明（return description）
- 例外說明（raises description）
- 範例區段（examples section）
- 非同步完成語意（async completion semantics）

## 風格格式術語

- reST 標記（`:param` / `:returns` / `:raises`）
- Google-style 區段（`Args` / `Returns` / `Raises`）

## 建議語氣

- 使用台灣繁體中文。
- 優先使用「回傳」「拋出」「逾時」「流程」「語意」等用詞。
- 避免低資訊句型（例如「執行某功能」「方法輸入參數」）。
